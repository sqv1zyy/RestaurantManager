Что бы запустить интерфейс:
1. В PgAdmin создаём новую базу данных и выполняем sql код
2. Открываем среду разработки
3. В терминале пишем: git clone https://github.com/sqv1zyy/inteface.git
4. Затем пишем pip install requirements.txt
5. В файле config.py указываем название базы данных, и пароль от сервера
6. в терминал пишем cd inteface, затем python main.py
